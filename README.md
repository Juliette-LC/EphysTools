These are a selection of tools I built and used over the years to analye and plot nanopore electropysiology data, mostly recorded from Orbit mini or Oxford Nanopore Technologies MinION devices.
These are all original tools, but I used Claude to comment and re-organise them to make it more accesible to other people.
